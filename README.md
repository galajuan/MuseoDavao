# Museo Davao 🏛️

Museo Davao is a digital web platform dedicated to preserving and showcasing the rich cultural, historical, and natural heritage of Davao City, Philippines. This project serves as a centralized, interactive archive for the community.

## Key Features
* Interactive Collection: Browse artifacts across categories like Indigenous Traditions, Colonial History, and Natural History.
* User Authentication: Secure sign-in and registration powered by Supabase.
* Shopping & Cart: Integrated online shop for museum souvenirs and replicas.
* AI-Powered Guide: An interactive chatbot to answer visitor questions.
* Responsive Design: Optimized for both desktop and mobile viewing with cultural design patterns.

## Technologies Used
* Frontend: HTML5, CSS3, JavaScript.
* Database & Auth: Supabase (PostgreSQL & Authentication).

## Project Structure
- /assets: Contains images, including the Hero.png background.
- /pages: Dedicated pages like collection.html, shop.html, events.html, and about.html.
- /css: Contains style.css for UI/UX styling.
- config.js: Supabase client initialization and API configurations.
- index.html: The main entry point for the website.

## Setup & Installation
1. Clone this repository to your local machine.
2. Set up a project in Supabase.
3. Update config.js with your Supabase Project URL and API Key.
4. Run the SQL schema for your tables in the Supabase SQL editor.
5. Open index.html in your browser or use a live server extension.
